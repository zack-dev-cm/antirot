# Security Policy

AntiRot is a local-first linting tool and does not ship a hosted service.

## Supported versions

Security fixes will be applied to the latest release line only.

## Reporting a vulnerability

If you believe AntiRot has a security issue, do not open a public GitHub issue first.

Send a private report with:

- affected version
- reproduction steps
- impact assessment
- any proposed mitigation

Until a dedicated security contact is set up, report through a private GitHub security advisory if the repository has that feature enabled after publish.

If that is not available, contact the maintainer directly through the GitHub profile associated with the repository.
